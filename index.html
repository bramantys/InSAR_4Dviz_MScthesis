Repository

Open GitHub repository

Template package

The Proto1_RUM_TEMPLATE/ folder contains the reusable package structure and pipeline notes. It is not meant to be opened as a finished viewer.

Local use

Online use requires no installation. For local use, download the repository and run a simple local server: 

python -m http.server 8000

Then open http://localhost:8000/.
